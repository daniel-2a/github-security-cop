# GitHub Security — Community of Practice

> ⚠️ **This repository contains intentionally vulnerable GitHub Actions workflows for educational purposes. Do not use these workflows in production.**

## What's Here

A collection of deliberately insecure GitHub actions and workflows demonstrating real-world supply chain attack vectors, including the patterns exploited in incidents like the Trivy compromise.

The `zizmor.yaml` workflow is an exception — it is secure and flags vulnerabilities in GitHub actions and workflows.
